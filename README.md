# PhysioAssist Deployment

This repository contains the PhysioAssist physiotherapy recovery plan generator.

## Live Demo
https://recover-plan.netlify.app/

## Features
- AI-powered recovery plan generation
- Multi-provider support (Gemini + HuggingFace)
- Mobile-first responsive design
- Professional PDF export
- Video exercise demonstrations

## Quick Start
1. Open `config.js` and add your API keys (optional)
2. Open `index.html` in a browser
3. Fill out the patient form
4. Generate recovery plan

## Deployment
This app is deployed on Netlify and auto-syncs with GitHub.
